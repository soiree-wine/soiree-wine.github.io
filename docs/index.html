concept

日本全国からセレクトしたワインと
ポテンシャルを引き出すペアリング

ソワレは日本ワインが主役になる、
日本ワインに特化したバー・ビストロです。
ブラインドテイスティングコンクール優勝経験を持つ店長が、
4,000本のストックの中から飲み頃のものだけ、
お客様や料理との最適なペアリングでご提供いたします。

渋谷区千駄ヶ谷・北参道の閑静な住宅街の一角で、
進化し続けている日本ワインの今、
そして新たな発見が皆様をお待ちしております。

producer

ワイナリーとの架け橋に

日本ワインの品質は急速に高まっており、
その魅力をもっと知ってほしいという想いがこの店を始めたきっかけです。
私自身が日本全国のワイナリーへ足を運び、そこで感じたこととあわせて、
お客さまに伝えていきたいと思っています。

そこでのネットワークを活かして、
ワイナリーの方々をお招きしたメーカーズディナーなどのイベントも開催し、
作り手との架け橋になることも私たちの目標のひとつです。
新たな体験として、ソワレで日本ワインをお楽しみください。

店長志田 渉
